# 🏡 House Price Prediction Using Linear Regression

## 📌 Overview

This project predicts house prices using the **Linear Regression** algorithm based on selected house features. It includes data visualization, model evaluation, and an interactive prediction system for estimating house prices.

---

## ✨ Features

* Predicts house prices using **Linear Regression**
* Uses selected house features for prediction
* Generates a **Correlation Heatmap** for feature analysis
* Visualizes the **House Price Distribution**
* Compares **Actual vs Predicted Prices**
* Evaluates model performance using **MAE**, **RMSE**, and **R² Score**
* Saves the trained model using **Joblib**
* Provides an interactive user input system for real-time house price prediction

---

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Run the project:

```bash
python house_price_prediction.py
```
